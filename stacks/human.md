# Human

| Feature            | Description                                                                                          |
| ------------------ | ---------------------------------------------------------------------------------------------------- |
| **Tenacious**      | Gain +1 Attribute Point. You must still follow the attribute cap when applying this Attribute Point. |
| **Unique Feature** | To further define your human, choose one feature from the following list at level 1.                 |

## Features

| Feature                     | PreReq                            | Description                                                                                                                                                          |
| --------------------------- | --------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Creative**                |                                   | *Repeatable.* Gain +6 Skill Points that you can spend on any skills with the crafting tag. With this feature, you may put up to +4 points into any individual skill. |
| **Determined**              |                                   | Gain **Determined (1/Rest):** When you are reduced to 0 HP, immediately regain 1 HP.                                                                                 |
| **Determined II**           | Determined                        | Determined becomes 1/Recoup.                                                                                                                                         |
| **Determined III**          | Determined II, Character Level 15 | Determined becomes 1/Catch Breath.                                                                                                                                   |
| **Fast Friends**            |                                   | Gain +6 Skill Points that you can spend on any skills with the influence tag. With this feature, you may put up to +4 points into any individual skill.              |
| **Offensive Rush**          |                                   | Gain **Offensive Rush (1/Turn):** When moving towards a hostile creature, you may move +10 ft for 0 AP.                                                              |
| **Knack**                   |                                   | *Repeatable.* Gain +4 to one skill of your choice.                                                                                                                   |
| **Many Interests**          |                                   | *Repeatable.* Gain +2 to four separate skills of your choice.                                                                                                        |
| **Pillar of the Community** | Character Level 5                 | Choose a profession for which you have no ranks. Gain the rank 1 benefit of that profession.                                                                         |
| **Scrappy**                 |                                   | Gain the **Exploit** advanced maneuver.                                                                                                                                  |
